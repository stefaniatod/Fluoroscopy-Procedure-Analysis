# Business Questions

This project analyzes fluoroscopy procedures to answer the following questions:

1. How many fluoroscopy procedures are included in the dataset?

2. Which procedure type has the highest average Dose Area Product (DAP)?

3. Which procedure type has the longest average fluoroscopy time?

4. How many procedures were performed for each procedure type?

5. Is there a relationship between fluoroscopy time and DAP?

6. Which operator performed the highest number of procedures?

7. What is the age distribution of patients?

8. Are there differences in radiation exposure between procedure types?
