# Fluoroscopy Procedure Analysis

## Project Overview

This project analyzes fluoroscopy procedures using Python and SQL to explore radiation dose metrics and identify patterns across different procedure types.

The project demonstrates a complete data analysis workflow, including data cleaning, exploratory data analysis, SQL queries, and interactive visualizations.

---

## Objectives

- Explore fluoroscopy procedure data
- Analyze radiation dose metrics (DAP and Air Kerma)
- Compare different procedure types
- Investigate the relationship between fluoroscopy time and radiation dose
- Practice SQL queries for data analysis
- Create interactive visualizations

---

## Tools Used

- Python
- Pandas
- Plotly
- SQLite
- Jupyter Notebook
